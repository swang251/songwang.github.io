I used Endnote as my first reference management software when I started my master study. I also tried a little bit Mendeley and saw how well it is intrinsically coupled Elsevier publications and the PDF editor. However, I finally decided to use Zotero mainly because it is an open-source software and could be more flexible.

Workflow

  • Adding Items to Zotero
  • Download PDF \(\rightarrow\) select the paper item \(\rightarrow\) right click: Attach New File (ZotFile)
  • Right click: Add Attachment: Attach Stored Copy of File: Select the Org template \(\rightarrow\) Rename File from Parent Metadata

Paper PDF Organization

Zotero

  • Preference \(\rightarrow\) Sync \(\rightarrow\) File Syncing \(\rightarrow\) Sync attachment files in My Library using Zotero/WebDAV
  • Preference \(\rightarrow\) Advanced \(\rightarrow\) Files and Folders \(\rightarrow\) Linked Attachment Base Directory
    • What is Linked Files, see here
    • Base directory
    • Using ZotFile for a linked-file workflow

ZotFile

ZotFile is a Zotero plugin to manage your attachments: automatically rename, move, and attach PDFs (or other files) to Zotero items, sync PDFs from your Zotero library to your (mobile) PDF reader (e.g. an iPad, Android tablet, etc.) and extract annotations from PDF files.

  • ZotFile Preferences \(\rightarrow\) Renaming Rules
  • ZotFile Preferences \(\rightarrow\) General Setting
    • Source Folder for Attaching New Files
    • Location of Files
      • Attach stored copy of file(s)
      • Custom Location:
        • Use subfolder defined by /%F

Note Organization

Markdown Here

Markdown Here is an extension for Chrome, Firefox, Safari, Opera, Thunderbird, and Postbox.

  • One can build the extension bundle from the source code and generate the .xpi file for Zotero
  • Then in Zotero, go to Tools \(\rightarrow\) Add-on \(\rightarrow\) Install Add-on From File...
  • When you edit the note, go to File \(\rightarrow\) Markdown Toggle or the hotkey Ctrl+Alt+M to switch between Plain text and Rendered Test

Org-mode

Org mode is for keeping notes, maintaining TODO lists, planning projects, and authoring documents with a fast and effective plain-text system.

I started to use Org mode in 2018 and gradually fall in love with it. To me, there is never a substitute of Org mode. I tried different markdown software like Quiver, Joplin, etc. They are good, (especially the Joplin) but they cannot replace Org mode. Also, considering how painful it is to keep notes in the Zotero-note, even with the Markdown Here. I started to look for a way to use Org mode for writing reading summaries and notes which can be integrated with Zotero. Zotxt is the solution.
